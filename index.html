Guia de configuração (5 min)
1
Acesse o Google Cloud Console
Abra console.cloud.google.com e crie um projeto novo (ex: "FinFlow").
2
Ative a Google Drive API
Menu → APIs e Serviços → Biblioteca → busque "Google Drive API" → Ativar.
3
Configure a tela de consentimento OAuth
APIs e Serviços → Tela de consentimento OAuth → Externo → preencha nome do app ("FinFlow") → Salvar. Escopo: adicione drive.file.
4
Crie as credenciais OAuth
Credenciais → Criar credenciais → ID do cliente OAuth → Aplicativo da Web. Em "Origens JavaScript autorizadas" adicione https://drive.google.com e https://localhost. Em "URIs de redirecionamento" adicione https://drive.google.com/.
5
Copie o Client ID
Copie o valor do campo "ID do cliente" (termina em .apps.googleusercontent.com) e cole no campo acima.
6
Adicione-se como usuário de teste
Na tela de consentimento OAuth → Usuários de teste → adicione seu e-mail Google. Isso é necessário enquanto o app estiver em modo de teste.
7
Salve o arquivo no Drive e abra de lá
Faça upload deste arquivo HTML para o Google Drive. Sempre o abra direto pelo Drive. Seus dados serão salvos automaticamente em finflow_data.json no Drive.
